Installation Guide

Welcome to the AI Image Generation website! This guide will walk you through the process of deploying the React application from the dist folder to your online hosting platform.

Prerequisites

  • Access to an online hosting provider with file upload capabilities (e.g., cPanel, FTP, or SSH access).
  • A domain name (optional, but recommended).
  • Basic knowledge of file uploads and hosting platform usage.

Step-by-Step Installation

1. Prepare Your Files

The build process has generated a folder named dist containing all the necessary files for deployment.

  1. Ensure that all files in the dist folder are intact. These are the optimized files for your website.
  2. If the dist folder is located inside a larger project directory, navigate to this folder and make it ready for upload.

2. Connect to Your Hosting Account

You’ll need to log into your hosting provider to upload the files. You can do this through:

  • cPanel File Manager (common for shared hosting)
  • FTP Client (e.g., FileZilla) with your hosting credentials
  • SSH (if your hosting provider supports it)

3. Upload the dist Folder Contents

  1. Navigate to the root directory of your website (often named public_html or www).
  2. Upload the contents of the dist folder (do not upload the folder itself, only the files within it) to the root directory.
  3. Once the files are uploaded, ensure that index.html is in the root directory, as this file serves as the entry point for your React app.

4. Configure Hosting Settings (if needed)

Some hosting services require a few additional configurations:

  • If there’s an .htaccess file in the dist folder, upload it to the root directory as well. This file helps with routing in single-page applications like React.
  • If you encounter 404 errors on page refresh, add the following code to your .htaccess file to ensure all routes redirect to index.html:
RewriteEngine On
RewriteCond %{REQUEST_FILENAME} !-f
RewriteCond %{REQUEST_FILENAME} !-d
RewriteRule ^(.*)$ /index.html [L]

5. Test Your Application

  1. Visit your domain to verify that the application is running as expected.
  2. Test the image generation feature to ensure the AI model is loading and generating images correctly.
Note: If your website requires an API key or environment variables, make sure to configure them as instructed. You may need to set these in the hosting dashboard or use a configuration file.

Troubleshooting

  • If your site is not loading correctly, check the Console tab in your browser’s Developer Tools to identify any errors.
  • If you see a blank page, ensure all assets and files (JavaScript, CSS, images) are correctly uploaded.
